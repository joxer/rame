Guide

this guide illustrate you how to use Rame

the html file are in /public_html. The server handle the request from browser to this folder and retrive the correct file

The request for method are handle with the string /method/ into the url and then colon ( : ) and the name of the method

the method are in Classes/Method.
you can create easily a method writing rake method:add You must first write in method.yml the name of the method and then do rake method:add You can also delete a method with rake method:remove. You must write the method to delete in method.yml. You can archive the version of your Method file with db:ver_add the erubis file are in /m/ . enjoy it ^o^
